## 추천 extension (for vscode)

**code runner** : 간단한 단축키로 경로 지정없이 여러 언어의 코드를 실행시켜 vscode 내에서 디버깅할 수 있다.

_shortcut_  
 `ctrl + alt + n` : 지정 코드 실행  
 `ctrl + alt + j` or `F1` : 실행시킬 코드의 언어 지정
